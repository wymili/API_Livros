Para criar uma API é necessário traçar um objetivo, possuir um URL base, endpoints 
e os recursos que queremos disponibilizar/funcionalidades

 O objetivo dessa API é: Disponibilizar a consulta, criação, edição e exclusão de livros.

 EndPoints: localhost/livros(GET)
            localhost/livros/id(GET)
            localhost/livros/id(PUT)
            localhost/livros(DELETE)
 
Recursos: Livros
